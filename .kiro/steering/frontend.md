# Frontend Development Guidelines

## UI and Component Rules

### Technology Stack
- Next.js with TypeScript for React development
- Tailwind CSS for styling
- Redux Toolkit for state management
- Component-based architecture

### Component Development
- Create reusable, modular components
- Follow React best practices and hooks patterns
- Implement proper prop validation
- Use TypeScript interfaces for component props
- Maintain consistent component structure

### Styling Guidelines
- Use Tailwind CSS utility classes consistently
- Follow responsive design principles
- Maintain consistent spacing and typography
- Implement proper color schemes and themes
- Ensure accessibility standards (WCAG compliance)

### State Management
- Use Redux Toolkit for global state management
- Keep component state local when possible
- Implement proper action creators and reducers
- Use proper typing for Redux state and actions

### Performance Optimization
- Implement proper code splitting
- Use React.memo and useMemo where appropriate
- Optimize images and assets
- Implement proper loading states
- Use proper caching strategies

### User Experience
- Implement proper loading indicators
- Provide clear error messages
- Ensure smooth navigation and transitions
- Implement proper form validation
- Follow accessibility best practices
